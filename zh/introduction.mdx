## 什麼是啟潤支付？

啟潤支付是專為**中國出海 AI Startup** 打造的收款平台。它提供標準化的方式來接收海外客戶的付款，讓你專注於產品開發，而非支付基礎設施。

## 工作原理

典型的整合流程包含四個步驟：

<Steps>
  <Step title="建立產品">
    使用產品 API 定義你的產品，包括名稱、價格和貨幣。
  </Step>
  <Step title="建立 Checkout Session">
    當客戶要付款時，建立一個 Checkout Session 並將客戶重新導向到支付頁面。
  </Step>
  <Step title="客戶完成支付">
    客戶在託管的收銀頁面上選擇支付方式（信用卡、微信支付等）並完成付款。
  </Step>
  <Step title="接收 Webhook 通知">
    啟潤支付向你的伺服器發送 Webhook 確認付款。據此更新你的系統。
  </Step>
</Steps>

| 方式 | 地區 | 貨幣 |

| 支付寶 | 中國、東南亞 | CNY、USD |
| PayNow | 新加坡 | SGD |
